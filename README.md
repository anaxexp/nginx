## Docker Images

‚ùóFor better reliability we release images with stability tags (`anaxexperience/nginx:1.14-X.X.X`) which correspond to [git tags](https://github.com/anaxexp/nginx/releases). We strongly recommend using images only with stability tags. 

Overview:

* All images are based on Alpine Linux
* Base image: [anaxexperience/alpine](https://github.com/anaxexp/alpine)
* [Travis CI builds](https://travis-ci.org/anaxexp/nginx) 
* [Docker Hub](https://hub.docker.com/r/anaxexperience/nginx)

Supported tags and respective `Dockerfile` links:

## Orchestration actions

Usage:
```
make COMMAND [params ...]

commands:
    git-clone [url branch]
    git-checkout [target is_hash]
    check-ready [host max_try wait_seconds delay_seconds]
 
default params values:
    host localhost
    max_try 1
    wait_seconds 1
    delay_seconds 0
    is_hash 0
    branch ""    
```
